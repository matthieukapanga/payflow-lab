# PayFlow Lab 

A hands-on DevOps home lab simulating a real-world payment processing infrastructure.

## Project Overview
PayFlow Lab is a personal project built to simulate what a Junior DevOps Engineer
works with daily — containerized applications, cloud infrastructure, CI/CD pipelines,
and production monitoring.

## Tech Stack
- *OS:* Ubuntu Linux (via VirtualBox)
- *Containerization:* Docker & Docker Compose
- *Cloud:* AWS (EC2, S3, RDS, CloudWatch, IAM, VPC)
- *Infrastructure as Code:* Terraform
- *CI/CD:* GitHub Actions
- *Monitoring:* Prometheus + Grafana
- *Language:* Node.js (TypeScript)

## Project Sessions
| Session | Topic |
